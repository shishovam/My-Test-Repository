# Функциональные требования

## 1. Цель проекта

Создать веб-приложение для отслеживания результатов матчей Английской премьер-лиги сезона 2026/2027 с автоматическим расчётом турнирной таблицы. Данные хранятся в облачной базе Cloudflare D1, доступ защищён авторизацией.

## 2. Основные функции

### 2.1 Авторизация
- **FR-001:** Вход в систему по логину и паролю (preset-учётная запись администратора)
- **FR-002:** Выдача JWT при успешном входе, хранение токена на клиенте
- **FR-003:** Защита всех операций с данными — доступ только при валидном JWT

### 2.2 Управление результатами
- **FR-004:** Добавление результата матча (тур, домашняя команда, гостевая команда, счёт, дата)
- **FR-005:** Просмотр матчей выбранного тура
- **FR-006:** Редактирование счёта и даты ранее введённого матча
- **FR-007:** Удаление ошибочно введённого результата

### 2.3 Турнирная таблица
- **FR-008:** Автоматический расчёт очков по правилам АПЛ (3 за победу, 1 за ничью, 0 за поражение)
- **FR-009:** Сортировка команд по очкам, разнице мячей, забитым мячам, затем по алфавиту
- **FR-010:** Отображение полной статистики: игры, победы, ничьи, поражения, забитые/пропущенные мячи, разница, очки
- **FR-011:** Мгновенный пересчёт таблицы после любого изменения матчей

### 2.4 Хранение данных
- **FR-012:** Все данные хранятся в облачной БД Cloudflare D1
- **FR-013:** Данные сохраняются между сессиями и доступны с любого устройства

## 3. Ограничения

- **LIM-001:** Один турнир — только Английская премьер-лига
- **LIM-002:** Один пользователь (администратор) — нет многопользовательского режима и регистрации
- **LIM-003:** Ручной ввод результатов — без автоматической загрузки из внешних источников
- **LIM-004:** 20 команд АПЛ сезона 2026/2027
- **LIM-005:** Максимум 380 матчей за сезон (38 туров)
- **LIM-006:** Восстановление пароля не предусмотрено — учётные данные заданы заранее (упрощение для снижения сложности)

## 4. Нефункциональные требования

### 4.1 Безопасность
- **SEC-001:** Пароль хранится только в виде хеша (PBKDF2 / SHA-256 через Web Crypto API), в открытом виде не хранится
- **SEC-002:** Доступ к API защищён JWT; токен подписывается серверным секретом
- **SEC-003:** Секреты (JWT-секрет) хранятся в защищённом хранилище Cloudflare (wrangler secret), не в коде и не в репозитории
- **SEC-004:** Все запросы выполняются по HTTPS (обеспечивается Cloudflare по умолчанию)
- **SEC-005:** Параметризованные запросы (prepared statements) для защиты от SQL-инъекций

### 4.2 Производительность
- **PERF-001:** Загрузка приложения менее 3 секунд
- **PERF-002:** Edge-выполнение Worker без cold start
- **PERF-003:** Мгновенный расчёт таблицы при добавлении результата

### 4.3 Удобство использования
- **UX-001:** Адаптивный дизайн для десктопа, планшета и мобильных устройств
- **UX-002:** Единый рабочий экран после входа (селектор тура, матчи, таблица) без лишних переходов
- **UX-003:** Интуитивно понятный интерфейс
- **UX-004:** Валидация вводимых данных для предотвращения ошибок

### 4.4 Совместимость
- **COMP-001:** Работа в современных браузерах (Chrome, Firefox, Safari, Edge)
- **COMP-002:** Не требует установки дополнительного ПО на стороне пользователя

### 4.5 Надёжность хостинга
- **REL-001:** Использование Cloudflare (Pages, Workers, D1), где бесплатный тариф не приостанавливает проекты за неактивность — для долгоживущего портфолио

## 5. Будущие возможности (вне текущего scope)

- Интеграция с внешним API для автоматического получения результатов
- Статистика по игрокам
- Календарь матчей
- История предыдущих сезонов
- Графики и визуализация данных
- AI-функции: предсказание результатов, генерация комментариев, чат-бот по статистике

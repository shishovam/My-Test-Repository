# Руководство пользователя

## 1. Вход в систему

### 1.1 Авторизация
1. Откройте приложение в браузере
2. На странице входа введите:
   - **Логин:** имя учётной записи администратора
   - **Пароль:** ваш пароль
3. Нажмите «Войти»
4. После успешного входа откроется главный экран

> Восстановление пароля не предусмотрено: учётные данные заданы заранее. Храните их в надёжном месте.

### 1.2 Сессия
- Сессия действует 24 часа, после чего потребуется войти заново
- Для выхода используйте кнопку «Выход» в правом верхнем углу
- На общих компьютерах обязательно выходите из системы после работы

## 2. Главный экран

После входа всё находится на одной странице и разделено на три секции:

1. **Селектор тура** — кнопки туров 1–38 для переключения
2. **Матчи выбранного тура** — список результатов и кнопка «Добавить матч»
3. **Турнирная таблица** — автоматически пересчитывается при любом изменении

### 2.1 Навигация по турам
- Нажмите на номер тура, чтобы открыть его матчи
- Активный тур подсвечен
- Секция матчей обновляется без перезагрузки страницы

## 3. Ввод результатов матчей

### 3.1 Добавление матча
1. Выберите нужный тур
2. Нажмите «Добавить матч»
3. В форме укажите:
   - **Домашнюю команду** (выпадающий список)
   - **Гостевую команду** (выпадающий список)
   - **Счёт** домашней и гостевой команд
   - **Дату матча**
4. Нажмите «Сохранить»

### 3.2 Правила ввода
- ⚠️ Нельзя выбрать одну и ту же команду как домашнюю и гостевую
- ⚠️ Счёт — целое число не меньше 0
- ⚠️ Все поля обязательны
- ⚠️ Одну и ту же пару команд нельзя добавить дважды в одном туре

## 4. Редактирование и удаление

### 4.1 Изменение результата
1. Найдите матч в списке тура
2. Нажмите кнопку редактирования (✏️)
3. Измените счёт или дату
4. Сохраните — таблица обновится мгновенно

### 4.2 Удаление матча
1. Нажмите кнопку удаления (🗑️) рядом с матчем
2. Подтвердите действие
3. Матч удалится, таблица пересчитается автоматически

## 5. Турнирная таблица

### 5.1 Столбцы
- **#** — позиция
- **Команда** — название
- **И** — сыграно
- **В / Н / П** — победы / ничьи / поражения
- **ЗМ / ПМ** — забито / пропущено
- **РМ** — разница мячей
- **О** — очки (3 за победу, 1 за ничью)

### 5.2 Порядок сортировки
1. Очки (больше — выше)
2. Разница мячей
3. Забитые мячи
4. Алфавитный порядок

### 5.3 Обновление
Таблица пересчитывается мгновенно при добавлении, изменении или удалении матча. Данные хранятся в облаке и доступны при следующем входе с любого устройства.

## 6. Решение проблем

### 6.1 Не получается войти
- Проверьте правильность логина и пароля
- Убедитесь, что Caps Lock выключен
- Восстановление пароля не предусмотрено — используйте заранее заданные учётные данные

### 6.2 Изменения не сохраняются
- Проверьте подключение к интернету
- Обновите страницу
- Убедитесь, что все поля формы заполнены корректно

### 6.3 Сессия истекла
- Если появилось требование войти заново — это нормально через 24 часа
- Войдите повторно, данные сохранены

### 6.4 Ошибка «Матч уже существует»
- В одном туре нельзя дважды ввести одну пару команд
- Проверьте, не добавлен ли матч ранее; используйте редактирование вместо повторного добавления

## 7. Рекомендации

- Вводите результаты сразу после окончания тура
- Проверяйте счёт перед сохранением
- Используйте современный браузер (Chrome, Firefox, Safari, Edge)
- Выходите из системы на общих компьютерах
